# 🌿 MindfulDay

A simple wellbeing tracker to help you stay mindful, build small habits, and feel good every day. Built with pure **HTML, CSS & JavaScript**.

---

## ✨ What it does

- 💭 Shows a calming thought of the day
- ✅ Lets you check off daily wellbeing habits
- 📊 Shows your progress in a clean circular tracker

---

## 🧰 Made with

`HTML` `CSS` `JavaScript` — no frameworks, no installs.
